## Local installation
1. Clone the repository
2. npm install
3. We connect our repository with the database
4. Execute the command npm run db-reset
5. npm run dev
## Endpoints
 路 End-point Para Mostrar Las Citas De Un Usuario
 ```
 http://127.0.0.1:3000/api/customer/viewDates/3
 ```
路 End-point Para Mostrar El Perfil De Un Usuario
```
http://127.0.0.1:3000/api/customer/viewMyProfile/3
```
路 End-point Para Hacer El Login
```
http://127.0.0.1:3000/auth/login
```
路 End-point Para Hacer El Registro
```
http://127.0.0.1:3000/auth/register
```
路 End-point Para Que Un Usuario Elimine Una Cita
```
http://127.0.0.1:3000/api/customer/deleteUserDates/1
```
路 End-point Para Que Un Usuario Modifique Una Cita
```
http://127.0.0.1:3000/api/customer/updateUserDate/1
```
路 End-point Para Que Un Usuario Admin Vea Todos Los Usuarios
```
http://127.0.0.1:3000/api/admin/viewUsersAdmin
```
路 End-point Para Que Un Usuario Admin Vea Todos Las Citas
```
http://127.0.0.1:3000/api/admin/viewDatesAdmin
```
路 End-point Para Que Un Usuario Cree Una Cita
```
http://127.0.0.1:3000/api/customer/newUserDate
```
